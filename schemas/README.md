# Schemas

This directory defines the product contracts between CaseBuilder stages.

## Contracts

- `capture-bundle.schema.json` - source material and provenance input.
- `canonical-case-record.schema.json` - verified derived record.
- `filing-package.schema.json` - generated output package.

## Pending

- review artifact contract.
- human approval artifact contract.

Schemas should remain implementation-neutral and stable enough for multiple tools to consume.
