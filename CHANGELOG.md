# [0.1.0] - 2026-06-29

## Project Founded

Initial product architecture for CaseBuilder.

CaseBuilder is introduced as a litigation operating system focused on transforming court records into verified case knowledge before any legal work product is generated.

## Added

### Product Vision
- Capture → Build → Review → Generate workflow.
- Recursive litigation lifecycle.
- Human review gate before generation.
